--- layout: post title: Raccoon, Apache sui cellulari Nokia date: '2006-06-19T21:21:00.002+02:00' author: Andrea Fortuna tags: - Hacking - Internet modified_time: '2012-04-26T14:23:57.523+02:00' blogger_id: tag:blogger.com,1999:blog-1948348952555789070.post-3030626859034050230 blogger_orig_url: http://www.andreafortuna.org/2006/06/raccoon-apache-sui-cellulari-nokia_7313.html --- Nokia sta lavorando a un port del web server Apache.
Il nome in codice scelto per la versione mobile di httpd e' Raccoon e permette di mettere in piedi un web server su un cellulare con sistema operativo Symbian.
E' stato realizzato anche il port di mod_python per consentire la visualizzazione di pagine generate dinamicamente da script Python.
Per permettere a chiunque navighi di raggiungere il mobsite (cosi' vengono definiti i siti in hosting sui cellulari) Nokia ha messo a punto un sistema in grado di ridirezionare le richieste attraverso un "gateway" che consente a qualsiasi client di raggiungere il sito web in maniera trasparente per l'utente finale.



E' possibile scaricare i sorgenti e provare ad allestire un gateway utilizzando un sistema Linux oppure attivare un account gratuito su un gateway Nokia.
