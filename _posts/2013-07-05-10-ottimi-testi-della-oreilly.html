La O'Reilly e' a mio avviso una delle migliore case editrici per quanto concerne l'IT in generale e la programmazione in particolare.

Sostiene apertamente l'OpenSource e la condivisione delle informazioni, tanto da permettere la lettura gratuita online di un buon numero di titoli gia' pubblicati ma anche di versioni 'beta' non ancora stampate.

Su linuxtoy.org trovo una interessante lista delle 10 migliori pubblicazioni liberamente consultabili.


La riporto integralmente:


Mastering Perl
 Git Pocket Guide
Vagrant: Up and Running
High Performance Browser Networking
802.11ac: A Survival Guide
Test-Driven Development with Python
Interactive Data Visualization for the Web
HTML5 Canvas
Programming JavaScript Applications
Agile Data
