La Beta 2 di VmWare server è estremamente lenta.

Questo accade perchè (essendo una Beta) ha i flag di debug abilitati: il meccanismo di debug quindi aumenta il carico di lavoro della macchina host.

Ecco una procedura per eliminare il Debug e  velocizzare il tutto.

Fermare la virtual machines:

/etc/init.d/vmware stop

Se sulle macchine virtuali non sono installati i VMWare Tool sara' necessario effettuare lo shutdown dei singoli sistemi virtualizzati.

Entrare nella cartella lib di vmware (/usr/lib/vmware).

Qui e' presente un folder bin-debug: rinominarlo con

mv /usr/lib/vmware/bin-debug /usr/lib/vmware/bin-debug.bak

Ora creare un simlink della directory bin chiamandolo bin-debug:

ln -s /usr/lib/vmware/bin /usr/lib/vmware/bin-debug

La directory bin contiene i binari epurati dal codice di debug.

Dopo aver creato il symlink, riavviare VMware Server:

/etc/init.d/vmware start
