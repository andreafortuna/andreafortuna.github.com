Come al solito, MySql viene rilasciato sotto due tipologie di licenza: i clienti possono scaricare il software gratuitamente in base alla licenza open-source General Public License (GPL) oppure pagare per una licenza commerciale.



La seconda opzione ?edicata soprattutto ai software vendor terze parti che vogliono rivendere MySQL con i rispettivi prodotti.

Vengono forniti inoltre una serie di servizi di supporto, battezzati MySQL Network, che variano in prezzo da 245 euro per server all'anno a 3.995 euro per server all'anno, a seconda del livello d'offerta.
