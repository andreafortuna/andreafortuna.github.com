--- layout: post title: Javascript, risparmiare codice date: '2006-12-31T14:06:00.001+01:00' author: Andrea Fortuna tags: - JavaScript modified_time: '2012-04-26T14:23:36.447+02:00' blogger_id: tag:blogger.com,1999:blog-1948348952555789070.post-5434108482702843231 blogger_orig_url: http://oldsite.andreafortuna.org/2006/12/javascript-risparmiare-codice_31.html --- Sviluppando per il web, e utilizzando molto il javascript per lavorare sul DOM si fa un larghissimo uso della funzione document.getElementById('elemento').

Per risparmiare spazio e battute sui tasti, ho iniziato a utilizzare un remap della funzione suddetta:
function $(elemento) {
return document.getElementById(elemento);
}

in questo modo sostituiamo document.getElementById('elemento') con $('elemento').

Un bel risparmio, vero? :)
