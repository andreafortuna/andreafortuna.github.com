--- layout: post title: Windows Vista Restart Manager date: '2005-12-05T11:12:00.001+01:00' author: Andrea Fortuna tags: - Microsoft modified_time: '2012-04-26T14:24:31.156+02:00' blogger_id: tag:blogger.com,1999:blog-1948348952555789070.post-6882243799126096469 blogger_orig_url: http://oldsite.andreafortuna.org/2005/12/windows-vista-restart-manager_05.html --- Sembra che Microsoft stia lavorando su una interessante funzionalita' del prossimo sistema operativo, Windows Vista. Restart Manager, questo e' il nome della feature, si occupera' dii riavviare parti del sistema e applicazioni senza richiedere il reboot totale della macchina. Afferma Jim Allchin (co-presidente per la divisione prodotti e servizi Microsoft):
"If a part of an application, or the operating system itself, needs to updated, the Installer will call the Restart Manager, which looks to see if it can clear that part of the system so that it can be updated. If it can do that, it does, and that happens without a reboot"
Se invece il sistema avesse bisogno di essere riavviato completamente, lo stato delle applicazioni e del sistema stesso verra' congelato, il computer riavviato e in seguito ripristinato lo stato dei programmi come prima del restart. A quanto pare, solo le applicazioni appositamente sviluppate potranno usufruire delle funzionalita' di Restart Manager.
