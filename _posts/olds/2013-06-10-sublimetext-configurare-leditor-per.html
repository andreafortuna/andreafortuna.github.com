Ultimamente sto utilizzando con discreta soddisfazione SublimeText, un editor leggero, multipiattaforma, ricco di funzionalità e estremamente personalizzabile.

Fornisce supporto a molti linguaggi, permette di gestire progetti e la loro relativa compilazione.

Una mancanza però l'ho notata: non permette di default la compilazione di progetti Node.js.

La lacuna si colma con pochi click.



Per prima cosa, da menù Tools, selezionare Build system e poi New Build System... :


Nel file che si aprirà, copiate questo testo:
{
 "cmd": ["node","$file"]
}
su OSX potrebbe essere necessario specificare il path completo del binario node:
{
 "cmd": ["/usr/local/bin/node","$file"]
}

Salvate il file (io ho optato per NodeJS.sublime-build):


A questo punto, nel menù Tools -> Build System sarà presente la nuova voce NodeJS:


Selezionandola e richiamando successivamente Build, il progetto verrà avviato:


Spero sia utile! :-)
