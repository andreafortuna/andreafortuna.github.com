Cito Franticindustries:
WebOS is a virtual operating system that runs in your web browser. More precisely, it’s a set of applications running in a web browser that together mimic, replace or largely supplement a desktop OS environment.

Con l'avvento del Web 2.0 sono nati molti progetti orientati alla realizzazione di un 'surrogato' di sistema operativo che giri in un browser.

In questo post da franticindustries.com vengono recensiti i 10 piu' promettenti.
