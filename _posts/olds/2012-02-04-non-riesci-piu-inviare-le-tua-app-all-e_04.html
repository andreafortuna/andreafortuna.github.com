Se il tool di validazione ti restituisce un errore di questo tipo:
iPhone/iPod Touch: Icon.png: icon dimensions (0 x 0) don’t meet the size requirements. The icon file must be 57x57 pixels, in .png format

allora puoi risolvere semplicemente installando l'ultima versione di Application Loader.
