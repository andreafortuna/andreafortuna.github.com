Installate un intero stack LAMP/WAMP/MAMP? Installate una istanza di Apache/Lighttpd?



Non serve, basta una riga di Python, utilizzando il modulo SimpleHTTPServer:

python -m SimpleHTTPServer

Il server rispondera' all'indirizzo http://localhost:8000 e servira' i file presenti nella directory dalla quale viene invocato.
