--- layout: post title: 'Windows 7: sessioni concorrenti di desktop remoto, come abilitarle' date: '2012-03-28T10:53:00.001+02:00' author: Andrea Fortuna tags: - windows 7 - remote desktop - Windows - Hacking - Microsoft modified_time: '2013-05-22T09:28:32.353+02:00' thumbnail: http://1.bp.blogspot.com/-dYsgYm4O8LY/UZxzjCsT77I/AAAAAAAAEw0/JwYj30il3hk/s72-c/rdp.png blogger_id: tag:blogger.com,1999:blog-1948348952555789070.post-8252399472512852506 blogger_orig_url: http://oldsite.andreafortuna.org/2012/03/windows-7-sessioni-concorrenti-di_28.html --- Windows 7 permette l'utilizzo del desktop da parte di un solo utente alla volta.

Ad esempio, se un utente sta utilizzando il PC e un'altro tenta di collegarsi via remote desktop, quello sulla postazione fisica viene immediatamente disconnesso.

Per fortuna i ragazzi di MissingRemote hanno rilasciato una semplice patch per rimuovere questa limitazione:

  1. Per prima cosa, scaricate il pacchetto contenente la patch

  2. Decomprimete il file (ad esempio in C:\Win7RDP), al suo interno saranno presenti i seguenti files: 32_rdpclip.exe,32_termsrv.dll,64_rdpclip.exe,64_termsrv.dll,Install.cmd

  3. Aprite un prompt dei comandi con privilegi di amministratore (Start -> Programmi -> Accessori -> Click destro su Prompt dei Comandi -> Esegui come amministratore)

  4. Dal prompt, recatevi nella cartella dove avete scompattato l'archivio (C:\Win7RDP)

  5. Scrivere Install e premere INVIO.
